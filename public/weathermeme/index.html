Weather Meme

Memes. For. The. Weather.

Github

About

Weather Meme is a (hopefully one day) collection of apps that provides a meme to the user that is relevant to the current weather conditions at the user's location. It's mostly just a project for me to learn various technologies.

Apps

I have made an API that allows apps to be easily created. This means that I can use this API to learn about whatever I want by making various implementations of this api. So if I feel like learning about Android development, I might make an Android WeatherMeme app. I might make a web app, a desktop app, or even a reddit bot that implements this API in order to learn whatever technology I feel like learning.
