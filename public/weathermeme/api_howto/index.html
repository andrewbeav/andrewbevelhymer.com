WeatherMeme Api

The API that relieves you on those days where you just want some dank weather related memes

How To Use

Using this API is simple, first you need to head over to Open Weather Map and get an API key for their API.

Once you do that, you will include it as a query to the request for this API. The structure for making a request to this API is:

http://andrewbevelhymer.com/weathermeme/api/?owm_appid=your_open_weathermap_api_key&lat=latitude&lon=longitude

The Object

This request, if everything is valid, will return a JSON that contains three things

meme_code

This is the same as the file name of the meme without the .png extension. Might be useful

meme_location

Location of the meme. You can use this to get the meme (via HTTP) in your application

weather_info

This is just the object that is given from Open Weather Map. It contains all the weather info for that particular location. Check their documentation for details.
